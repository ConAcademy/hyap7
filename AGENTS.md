# hya-7

`hya-7` is a TUI for working with the quantitative ethical question "Half your age minus 7"

It is a Golang application using [BubbleTea](https://github.com/charmbracelet/bubbletea) for TUI and [NTCharts](https://github.com/NimbleMarkets/ntcharts) for charting.

I have some ideas in mind, but I'd be interested in how you think it should work.
